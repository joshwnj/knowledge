Say you have a file called `.env/dev`, with environment vars in it like this:

```
# here is a comment
PORT=8000

# here is another comment
SECRET=password123
```

Little shortcut to populate your environment:

```
export $(cat .env/dev | grep -v ^# | xargs)
```
